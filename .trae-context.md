# Protope 项目上下文

## 技术栈
- React 19.2.6
- Vite 8.0.12
- TypeScript 6.0.2
- React Router DOM 7.16.0 (createBrowserRouter)
- Ant Design 6.4.3
- dayjs 1.11.21

## 项目结构
```
src/
  router/index.tsx      # 路由配置，使用 createBrowserRouter
  layout/Layout.tsx     # 全局布局组件
  pages/
    A2ATool.tsx         # A2A工具页面
    SkillCenter.tsx     # 技能中心页面
  assets/               # 静态资源
```

## 开发规范
- 使用函数组件 + Hooks
- TypeScript 严格模式
- 路由使用 React Router DOM v7 新API
- UI组件优先使用 Ant Design
